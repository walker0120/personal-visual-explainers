# 薬の専門用語 → やさしい言葉 変換辞書

## 変換の原則

1. **患者さんが聞いたことのある言葉で言い換える**: 難しい薬理学用語はなるべく使わない
2. **身体の中で何が起きるかを具体的に**: 「ブロックする」「抑える」だけでなく体感できるたとえを使う
3. **初出時に解説ボックス**: 最初に出てきたとき必ず説明
4. **怖さを和らげる**: 副作用は「こういう体験をしたら教えてください」という形で伝える

---

## よく使う変換表

### 薬の基本

| 医療用語 | やさしい言葉 | 患者さんへのたとえ |
|---------|------------|-----------------|
| 有効成分 | 薬の主な働きをする成分 | カレーのスパイスのうち「辛さ」の部分 |
| 先発品 | 最初に作られたオリジナルの薬 | ブランド品 |
| 後発品（ジェネリック） | 同じ有効成分で作られた薬 | ブランド品と同じ素材を使ったノーブランド品 |
| 半減期 | 薬の量が半分になる時間 | 氷が溶けていくスピード |
| 血中濃度 | 血液の中にある薬の量 | 血液の中を流れる薬の濃さ |
| 剤形 | 薬の形（錠剤・カプセルなど） | 薬の「形」 |
| 徐放製剤 | ゆっくり溶けて効き目が長続きする薬 | 時間差で効くカプセル |

### 作用・副作用

| 医療用語 | やさしい言葉 | 患者さんへのたとえ |
|---------|------------|-----------------|
| 主作用 | 薬の本来の目的の働き | 風邪薬なら「熱を下げる」働き |
| 副作用 | 薬の意図しない働き | 目的の効果以外に身体が受ける影響 |
| 相互作用 | 薬同士が影響し合うこと | 食べ合わせが悪い食品の組み合わせ |
| 禁忌 | 絶対に飲んではいけない場合 | 「一緒に飲むと危険な組み合わせ」 |
| 慎重投与 | 条件付きで注意しながら使う場合 | 「念のため気をつけながら使う」 |
| 耐性 | 薬が効きにくくなること | 慣れてしまって効かなくなること |
| 離脱症状 | 薬を急にやめたときの体の反応 | 急にやめると体がびっくりして起きる症状 |

### 循環器・血圧

| 医療用語 | やさしい言葉 | 患者さんへのたとえ |
|---------|------------|-----------------|
| 降圧薬 | 血圧を下げる薬 | 水道管の水圧を調整する弁 |
| Ca拮抗薬 | カルシウムの働きをブロックして血管を広げる薬 | 血管の筋肉をゆるめて道を広くする |
| ARB | 血圧を上げるホルモンの受け口をふさぐ薬 | 「血圧を上げろ」という信号を届けないようにする |
| ACE阻害薬 | 血圧を上げる物質が作られるのを防ぐ薬 | 血圧を上げる命令を出す工場を止める |
| 利尿薬 | 尿の量を増やして血圧を下げる薬 | 余分な水を外に出して血管への圧力を減らす |

### 糖尿病・血糖

| 医療用語 | やさしい言葉 | 患者さんへのたとえ |
|---------|------------|-----------------|
| 血糖値 | 血液の中の糖分の量 | 血液の中の「砂糖の濃さ」 |
| インスリン | 血糖値を下げるホルモン | 細胞に糖を取り込むための「鍵」 |
| インスリン抵抗性 | インスリンが効きにくくなること | 鍵が錆びてドアが開きにくくなること |
| 低血糖 | 血糖値が下がりすぎること | 血液の中の糖分が足りなくなること |
| HbA1c | 過去1〜2ヶ月の血糖値の平均 | 血糖コントロールの「通知表」 |

### コレステロール・脂質
