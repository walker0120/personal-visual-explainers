# 薬の用語→患者向け言い換え辞書

## 変換の原則

1. **専門用語をそのまま使わない** — 患者は薬学を知らない前提で書く
2. **初出時は括弧書きで補足** — 「血圧を下げる薬（降圧薬）」のように
3. **数字・期間で具体化する** — 「まれ」より「100人に1〜2人」の方が伝わる

---

### 用法・用量

| 専門用語 | 患者向け | 補足 |
|---------|---------|------|
| 空腹時（食前30分） | 食事の30分前 | 「食べる前に飲む」 |
| 食直前 | 食事の直前 | 「食べる直前（5〜10分前）に飲む」 |
| 食後30分以内 | 食後すぐ | 「食べた後、なるべくすぐ飲む」 |
| 就寝前 | 寝る前 | 「寝る30分〜1時間前に飲む」 |
| 頓服 | 症状が出たときだけ | 「痛いとき、必要なときだけ飲む」 |
| 漸増 | 少量から増やす | 「最初は少ない量から始めて、徐々に増やす」 |
| 中断禁止 | 自己判断でやめない | 「調子が良くなっても、自分でやめないでください」 |

| 専門用語 | 患者向け | 補足 |
|---------|---------|------|

## よくある説明フレーズ集

患者向けの説明文を書くときに使えるフレーズ。

| 目的 | フレーズ例 |
|-----|----------|
| 副作用を和らげる | 「気になる症状が続く場合は、次回の受診時に相談してください」 |
| 服薬継続を促す | 「症状がよくなっても、自分の判断でやめないでください」 |
| 緊急受診を促す | 「次の症状が出たらすぐに病院・薬局に連絡してください」 |
| 飲み忘れへの対応 | 「飲み忘れに気づいたら、すぐ飲んでください。次の服用時間が近い場合は1回分飛ばしてください。絶対に2回分を一度に飲まないでください」 |
| 相互作用の注意 | 「他の薬・サプリ・市販薬を飲む場合は必ず薬剤師に相談してください」 |
